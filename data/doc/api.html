get /suggestions

Fulltext autosuggester.

  • Request
  • Response

Query Parameters

  • query: required (string)

    Example:

    akusticke

Body

Type: application/json

Example:

{
  "meta": {
    "params": {
      "query": "akustic"
    }
  },

  "data": {
    "queries": [
      "akusticka basa",
      "akusticka pena",
      "akustika"
    ],
    "categories": [

        "str_id": "akusticke-a-klasicke-kytary",
        "name": "Akustické a klasické kytary",

      {
